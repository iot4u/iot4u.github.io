OpenHAB. Часть №1

Июль 13, 2015

*Что-то давно ничего не происходило в этом блоге. Исправляю.*

Пришло время осветить одно из программных решений в области IoT. Встречайте: OpenHAB - Open Home Automation Bus - само название подразумевает, что данное решение предполагается использовать для интерграции различных устройств в единое пространство, при постронии домашней(и не только) “сети”.

Если коротко описывать OpenHAB то стоит сказать, что это написанное на Java, полностью opensource программное решение, в основе которого лежит OSGI расширяемая архитектура( если вы понимаете о чем я … :-) ) Весь кода лежит на github.

Уже идет разработка OpenHAB2, но и первая версия не стоит на месте и освещать я буду именно её. В первой части я расскажу как запустить OpenHAB на linux системе в demo режиме, чтобы можно было попробовать как это все выглядит.

Вот что нам предстоит сделать:
0. Скачивание и установка jdk
1. Создание нового пользователя
2. Скачивание и распаковка openhab-runtime
3. Скачивание и распаковка openhab-demo
4. Создание конфигурационного файла
5. Первый запуск

И так:

Скачивание и установка jdk

Java следует взять с официального сайта, хотя можно использовать и openjdk, который можно найти в пакетах для любого linux дистрибутива. Если вы планируете использовать Raspberry Pi в качестве центрального узла вашей сети, то и для него есть специальная jdk для ARM архитектуры. Взять его можно здесь Не забываем про export JAVA_HOME=/usr/java/jdk1.8.0_33 и export PATH=$PATH:$JAVA_HOME/bin.

Создание нового пользователя

Не лишено смысла использовать для работы OpenHAB специального пользователя. Создать его можно так: sudo useadd -m openhab

Скачивание и распаковка openhab-runtime

Контейнер с OpenHAB распространяется в платформа независимом zip архиве и называется он openhab--runtime.zip. Его можно распаковать в /opt/openhab, надо только не забыть дать права пользователю openhab. В таком же zip-архиве распространяются многочисленные расширения и биндинги, но о них другой части.

Скачивание и распаковка openhab-demo

Демо настройка скачивается таким же архивом(openhab-demo), распаковывается и содержимое архива перемещается в соответствующие папки установленного на предыдущем шаге runtime (в /opt/openhab, не забываем про права)

Создание openhab.cfg

Практически всё готово к запуску, не хватает только файла конфигурации. Но и здесь за нас уже все сделали разработчики openhab. В каталоге /opt/openhab/configurations лежит файл openhab_default.cfg с настройками по умолчанию, остается только скопировать этот файл: cp openhab_default.cfg openhab.cfg. Подробно разберем файл конфигурации в другой части.

Всё готово к первому запуску. :-)

Первый запуcк

Для запуска OpenHAB: su - openhab -c /opt/openhab/start.sh !Первый запуск Openhab занимает порядка 1 минуты. Наберитесь терпения (… Java …). Теперь в браузере переходим по адресу: http://<openHAB address or hostname>:8080/openhab.app?sitemap=demo и наслаждаемся тем, что называется Classic UI.
